# 💳 Online Fraud Detection System

## 📌 Overview
This project is a Machine Learning-based system that detects fraudulent online transactions.  
It uses a Random Forest Classifier and provides real-time predictions through a Streamlit web application.

---

## 🚀 Features
- Real-time fraud prediction
- User-friendly web interface
- High accuracy using Random Forest
- One-hot encoding for transaction types
- Displays prediction confidence

---

## 🛠️ Tech Stack
- Python
- scikit-learn
- pandas
- numpy
- Streamlit

---

## 🤖 Model Details
- Algorithm: Random Forest Classifier
- Evaluation Metrics:
  - Accuracy
  - F1 Score
  - Confusion Matrix
